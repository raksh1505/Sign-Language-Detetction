# Sign-Language-Detetction
An integrated Machine Learning system designed to detect and translate American Sign Language(ASL) in real time. This project Random forest classifier for high-accuracy gesture recognition.

Key Features:
Real-time Detection: Process video frames to identify hand signs instantly.
ML Powered: Built using Python, Scikit-Learn, and OpenCV.
High Accuracy: Trained on diverse ASL datasets to ensure robust translation.

Tech Stack:
Language: Python
Libraries: Mediapipe, OpenCV, Scikit-Learn, NumPy
